Features:

  • Posts are converted to images and arranged into HTML pages.
  • This site publishes one page for each Locale & Category per day.
  • Locale & Category are case-sensitive.
  • The date format is YYYYMMDD where MM is from 01 to 12.
  • The date of the post is the timestamp of the block that includes the transaction.
  • Publishing is done soon after midnight UTC daily.

  • Fees:
    • The pay-to-post model keeps this site uncensorable.
    • Image Post: 1 finney per image which is 0.001 ETH
    • Text Post: 5 szabo per byte, which would be 0.00085 ETH for a 170-byte post.
    • Anyone can view the fees in the smart contract.
    • The contract owner can adjust the fees any time.
    • Fees cited are the required minimums. Senders may offer higher fees, but no change will be given.
    • No user accounts are maintained.

  • Network Miner Fees ("gas"):
    • The gas requirement depends on the quantity of data provided in the function call parameters.
    • Offer at least 40,000 gas for full 170-byte text posting.
    • Offer at least 35,000 gas for an image link posting.
    • Check ETH Gas Station for the current gas price when posting.
    • Ethereum returns the extra gas if one sends too much.
    • Sending insufficient gas fails the transaction and the gas is not returned to the sender.

Image Posts:

  • This site uses the posted link's url value as the src attribute within a <img> tag.
  • This site makes no attempt to validate links exist or preview resources prior to publishing.
  • This site does not save images that have been submitted.
  • Posters may choose IPFS to share images. See Public IPFS Gateways.
  • Tag width and height attributes will scale each image to 320x200 pixels.
  • We suggest using an image format that common browsers recognize, such as PNG, JPG, GIF, BMP, SVG.

Text Posts:

  • Each text post is converted to a 320x95 pixel PNG image.
  • Characters are placed in a logical grid 5 rows x 34 cols for a total of 170 visible. Extra characters are truncated.
  • The font used is Monospace Bold 12, black.
